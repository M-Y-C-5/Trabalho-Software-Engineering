# Trabalho-Software-Engineering
Trabalho para a faculdade UniFecaf para testar minhas habilidades na disciplina de Software Engineering.

O objetivo do projeto será criar um sistema de gerenciamento de tarefas, visando criar o sistema priorizando qualidade e eficiência a metodologia utilizada será a de métodos ágeis para manter a coesão e coerência entre a equipe e garantir um trabalho melhor produzido, em questão de integridade do código e funcionamento simples e direto.
